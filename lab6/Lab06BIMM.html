}

Key-points Every function in R look fundamentally the same in terms of its structure.. Name, input, and body

name  <- function(input) {
body
}

Functions can have multiple inputs. These can be required arguments or *optional** arguments having a set default value.

Q. Modify and improve our generate_dna() function to return it’s generated sequence in a more standard format like “AGTAGTA” rather than “A”, “C”, “G”, “A”

generate_dna <- function(len, fasta = TRUE) {
  
  ans <- sample(x = c("A", "C", "T", "G"), 
                size = len, 
                replace = TRUE)
  
  if(fasta) {
    ans <- paste(ans, collapse = "")
  }
   return(ans)
}
generate_dna()
[1] "AGAG"

A protein generating function

Q. Write a function called generate_protein(), that generates a user specificed length protein sequence.

Q. Use that function to generate random protein sequences between length 6 and 12

Q. Are any of your sequences unique i.e. not found anywhere in nature?

generate_protein <- function(len) {
  aa <- c("A", "R", "N", "D", "C", "Q", "E", "G", "H", "I", "L", "K", "M", "F", "P", "S", "T", "W", "Y", "V")
  ans <- sample(aa, size = len, replace = T)
  ans <- paste(ans, collapse = "")
  return(ans)
}
